Introduction

jQuery嚴格上來說並不是一種程式語言，而是一套遊覽器的JavaScript開源式函式庫。 第一個版本於2006年1月由John Resig在BarCamp NYC上釋出第一個版。 jQuery的設計目的是簡化原生Javascript的語法。

History

2005年8月，美國軟體工程師John Resig在個人部落格上提出簡化方式並改進Javascript中Prototype的“Behaviour”函式庫。

通過上述三個例子可以明顯的看對比出改善之後的語法更為簡潔。 這篇文章在發表後也引起了廣泛的關注，並於2006年1月14日正式發布以jQuery為名的函式庫。 這時的jQuery只支持了css selector, event 和 AJAX等事件的處理。 從2006年開始，jQuery從1.0版本更新至現今的2.2.3版本，也新增了許多非常方便實用的功能， 也讓它成為全世界最受歡迎的Javasricpt Library。
