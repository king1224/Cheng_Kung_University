*****程式註解*****


movave是用來做動平均
在fin中，我以五天為一周期(取太高會太平滑)


fin主要用來呼叫各方程式
然後作圖
其中呼叫lineout


lineout呼叫500次line2
做第1筆資料到第500筆資料的預測


line2中用了均線、MACD線、KD線方程式來計算預測(請見參考資料)
我取快線與慢線的最大差值為分母，去計算現在快線慢線'相對'遠近
但若取好幾年來的最大差值當分母會太過極端
故我取前30筆資料的差值最大(測驗後這樣最準)
其中有保護機制，若最大值小於0或最小值大於0 => 快慢線異常
=>快慢線沒有交叉，故不參考
所以取均線、MACD線、KD線的有效時候的值
輸出即為單次的漲幅、跌幅預測
